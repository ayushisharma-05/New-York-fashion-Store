# New York Fashion Store

New York Fashion Store is a multi-page static website built using HTML and CSS.  
The project focuses on layout design, structured navigation, dropdown menus, and section-based styling.

## Features

- Multi-page website structure
- Navigation bar with dropdown menu
- Banner and hero section
- Product showcase layout
- Image gallery section
- Newsletter subscription UI
- Contact section with embedded Google Map
- Footer with branding

## Tech Stack

- HTML5
- CSS3

## Project Structure

- index.html
- about.html
- collection.html
- gallery.html
- products.html
- contact.html
- signin.html
- account.html
- style.css
- Public (assets folder)
